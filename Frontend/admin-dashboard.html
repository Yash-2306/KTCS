💡 Recruiter Authentication Guide

This analytics panel aggregates live, database-backed traffic counts from both the public-facing landing page and secured dashboards. It verifies system authenticity using stateless server counters.

How It Works:

  • Whenever a user lands on the school site home page, an asynchronous POST hit request is registered.
  • Accessing the dashboards triggers a separate secure API logging hit.
  • All data is saved in a persistent database (MySQL) via Spring Boot JPA using transactional upsert updates to prevent concurrency bottlenecks.
